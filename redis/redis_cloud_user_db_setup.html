5
Add to application.properties
Paste credentials into your Spring Boot config
# application.properties
spring.redis.host=redis-12345.c1.us-east-1-2.ec2.cloud.redislabs.com
spring.redis.port=12345
spring.redis.password=your_password_here
spring.redis.ssl=true
✓ SSL must be true for Redis Cloud connections
6
Annotate your User Entity
Add @RedisHash to map User to Redis
// User.java
@RedisHash("User") ← stores as User:{id}
public class User implements Serializable {
  @Id
  private String id;
  private String name;
  private String email;
  @TimeToLive
  private Long expiration = 600L; ← 10 min TTL
}
✓ Each User stored as → User:101, User:102...
7
Create Redis CrudRepository
Extends CrudRepository for free CRUD operations
// UserRedisRepository.java
@Repository

    extends CrudRepository<User, String> {
  // save() findById() delete() — all free!
}
